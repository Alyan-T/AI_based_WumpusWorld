## Wumpus World: Knowledge-Based Logic Agent
A modern, browser-based implementation of the classic Wumpus World AI environment.

Key Features:

1. Logic Engine: Built-in Resolution Refutation engine that proves safe cells and identifies dangers using propositional logic.
2. Auto-Play Agent: Watch the AI autonomously navigate the grid based purely on its Knowledge Base (KB).
3. Real-Time Metrics: Live tracking of inference steps, visited cells, safe cells, and active KB clauses.
4. Customizable Grid: Adjust grid size and the number of pits on the fly.
5. Tech Stack: Vanilla JavaScript, HTML5, CSS3, and Bootstrap 5 (No heavy frameworks required).
